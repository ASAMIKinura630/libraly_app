DB(ERD) — エンティティ・リレーションシップ・ダイヤグラム

図書館保守システムで使用する全7テーブル（＋認証）の相関図

テーブル一覧（全7）

  • libraly_app_category — 図書カテゴリマスタ
  • libraly_app — 図書マスタ
  • libraly_app_faculty — 学部マスタ
  • libraly_app_department — 学科マスタ
  • libraly_app_student — 学生マスタ
  • libraly_app_staff — 担当者マスタ
  • libraly_app_lending_history — 図書貸し出し履歴

主なリレーション

  • カテゴリ → 図書（1:N、削除時 SET NULL）
  • 学部 → 学科 → 学生（1:N、RESTRICT）
  • 図書・学生 → 貸出履歴（1:N、RESTRICT）
  • 担当者 → 貸出履歴（貸出・返却処理者、RESTRICT）
  • auth.users ↔ 担当者（email 一致、論理紐づけ）
