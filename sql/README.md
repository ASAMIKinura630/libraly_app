## よくあるエラー

### `column "faculty" does not exist`

**原因:** `20260525_04` 実行後、学生テーブルから `faculty` / `department` 列が削除されているのに、古い INSERT（02/03）を実行した。

**対処:** リポジトリ最新の `20260525_02` / `20260525_03` を使う（`department_id` 形式）。04 より先に 02/03 を流す必要はありません。

### `department_id が NULL の学生が残っています`

**原因:** 04 実行時、学部・学科名がマスタと一致しない学生データがあった。

**対処:** マスタに学部・学科があるか確認し、不一致データを修正してから 04 を再検討。
